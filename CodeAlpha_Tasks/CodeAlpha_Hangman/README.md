## Hangman Game

This is a simple console-based Hangman game where players attempt to guess a hidden word one letter at a time.

## Features

* Selects a random word for every round
* Limits the number of guesses available
* Displays the word’s progress after each attempt

## Concepts Used

* `random` module
* Loop constructs
* String and list manipulation

## How to Run

python hangman.py
