## Chatbot

This project is a simple rule-based chatbot created using Python. It engages with users by giving predefined responses, making it a good starting point for learning basic conversational logic.

### Features

* Replies to greetings such as "hello" and "hi"
* Manages simple user questions
* Delivers preset responses
* Supports an interactive command-line interface

### Concepts Used

* `if-elif` conditional statements
* `while` loops for continuous execution
* Functions for organized code structure
* Basic string processing

### How to Run

* Ensure Python is installed on your system
* Run the following command in your terminal:
  `python chatbot.py`

### Example Interaction

You: hello
Bot: Hi there! How can I assist you?

You: how are you
Bot: I'm just a bot, but I'm doing well!
