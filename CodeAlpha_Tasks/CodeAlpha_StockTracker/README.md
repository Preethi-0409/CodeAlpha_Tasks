## Stock Portfolio Tracker

This is a simple project that calculates the total value of investments based on user input and preset stock prices.

## Features

* Lets users input stock names and their quantities
* Computes the total investment amount
* Easy to use and understand

## Concepts Used

* Dictionaries
* Input/output handling
* Basic calculations

## How to Run

python stock_tracker.py
